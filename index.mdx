      <Step title="Deploy with Docker">
        Run DeepLX instantly with Docker:
        
        ```bash
        docker run -d -p 1188:1188 ghcr.io/owo-network/deeplx:latest
        ```
        
        Your translation API is now live at `http://localhost:1188`
      </Step>

      <Step title="Make your first translation request">
        Send a POST request to translate text:
        
        ```bash
        curl -X POST http://localhost:1188/translate \
          -H "Content-Type: application/json" \
          -d '{
            "text": "Hello, world!",
            "source_lang": "EN",
            "target_lang": "DE"
          }'
        ```
        
        <Accordion title="Example Response">
        ```json
        {
          "code": 200,
          "id": 8359926003,
          "data": "Hallo, Welt!",
          "alternatives": ["Hallo Welt!", "Servus Welt!"],
          "source_lang": "EN",
          "target_lang": "DE",
          "method": "Free"
        }
        ```
        </Accordion>
      </Step>

      <Step title="Explore advanced features">
        DeepLX supports multiple deployment methods, authentication, proxy configuration, and more. Check out the [Configuration](/configuration/environment-variables) guide to customize your setup.
